## 清单

### Chrome

Google Chrome 是一款由谷歌公司开发的快速、安全且全球使用最广泛的跨平台网页浏览器。

> 下载链接：[Chrome](https://www.google.com/intl/zh-CN/chrome/)

![](https://img.lunamyth.love/2026/03/281c9cce6c412517f113c577a9bf11e8.jpg)

### Geek

Geek（Geek Uninstaller）是一款极其轻量且免安装的 Windows 卸载神器，它的核心作用是干净彻底地卸载软件，并强力扫描、清除常规卸载后留下的所有残留文件和注册表垃圾。

> 下载链接：[Geek](https://geekuninstaller.com/download)

![](https://img.lunamyth.love/2026/03/df88088e09777613240005019631ef17.jpg)

### Everything

Everything 是一款极致轻量的 Windows 本地搜索神器，它能通过毫秒级的索引速度，让你仅凭输入关键词就能瞬间全盘秒搜出所需的任何文件或文件夹。

> 下载链接：[Everything](https://www.voidtools.com/zh-cn/)

![](https://img.lunamyth.love/2026/03/ef11f14e9ba198a189eea75eab2ff729.jpg)

### Potplayer

PotPlayer 是一款性能强悍且极其轻量的 Windows 全能影音播放器，它内置了海量解码器，让你无需折腾额外插件，就能以极低的系统资源消耗流畅播放几乎所有格式的音视频文件。

> 下载链接：[Potplayer](https://potplayer.tv/?lang=zh_CN)

![](https://img.lunamyth.love/2026/03/a3f916a72023f2c50e19f8f3d2dc0929.jpg)

### Snipaste

Snipaste 是一款轻量且无广告的 Windows/Mac 屏幕截图与贴图神器，它的核心作用是精准捕捉屏幕任意区域，并将其悬浮“贴”在桌面上，极大地提升多任务处理和信息对照的效率。

> 下载链接：[Snipaste](https://zh.snipaste.com/download.html)

![](https://img.lunamyth.love/2026/03/60939657ca2814a11ccd3e30bf56151a.jpg)

### NanaZip

NanaZip 是一款基于 7-Zip 核心的现代开源解压缩神器，它专为 Windows 11 深度优化并完美原生融入了全新的系统右键菜单，提供轻量、极速且纯净无广告的文件解压与打包体验。

> 下载链接：[NanaZip](https://github.com/M2Team/NanaZip/releases)

![](https://img.lunamyth.love/2026/03/3e1b9e6b76137d69668278017e4c7804.jpg)

### Kaspersky

Kaspersky（卡巴斯基）是一款全球顶尖的老牌安全防护软件，它凭借极其强悍的病毒查杀引擎与多层主动防御技术，为设备提供抵御恶意软件、勒索病毒及各类网络攻击的全方位硬核守护。

> 下载链接：[Kaspersky](https://www.kaspersky.com.cn/)

![](https://img.lunamyth.love/2026/03/be93b2ec4427e351640f58bc88e7a2d5.jpg)

### VMware Workstation

VMware Workstation 是一款极其强大的专业级桌面虚拟机软件，它能在单台物理机上无缝、高性能地运行多个完全隔离的操作系统，是开发者和极客搭建测试环境或折腾跨平台操作的终极沙盒神器。现已对个人用户免费开放。

> 下载链接：[VMware Workstation](https://support.broadcom.com/group/ecx/productdownloads?subfamily=VMware%20Workstation%20Pro&freeDownloads=true)

![](https://img.lunamyth.love/2026/03/162d7572231ae2c172bac26157578369.jpg)

### MAS

Microsoft Activation Scripts（MAS）是一款开源、极简且绝对纯净的激活神器，它通过数字许可证 (HWID) 等安全机制，让你只需执行简短的命令脚本，即可一键实现 Windows 系统和 Office 办公套件的永久激活。

> 下载链接：[MAS](https://massgrave.dev/genuine-installation-media)

![](https://img.lunamyth.love/2026/03/02feb12cbef0bc9960aad2941f1b7507.jpg)

### Mocreak

Mocreak 是一款免费纯净且支持高度自定义的 Office 及 Windows 一键部署利器，它通过极简的图形界面，让你能按需选择组件、更改安装路径，全自动完成微软官方正版办公套件的极速下载与安装。

> 下载链接：[Mocreak](https://www.mocreak.com/download.html)

![](https://img.lunamyth.love/2026/03/a2b0631c22179307ba7cd35a993f3084.jpg)

### VSCode

Visual Studio Code (VSCode) 是一款免费开源、跨平台且极其轻量的现代代码编辑器，它凭借海量的扩展插件生态和强大的自定义能力，能瞬间变身支持几乎所有主流编程语言的顶级开发利器。

> 下载链接：[VSCode](https://code.visualstudio.com/)

![](https://img.lunamyth.love/2026/03/d4bad9dff80283fdeff408075f1d2cf3.jpg)

### TranslucentTB

TranslucentTB 是一款极度轻量的 Windows 任务栏美化小工具，它能一键将系统底部的任务栏变成全透明、毛玻璃或亚克力等质感效果，瞬间拉满整个系统桌面的视觉颜值与沉浸感。

> 下载链接：[TranslucentTB](https://translucenttb.net/#download)

![](https://img.lunamyth.love/2026/03/4480d640a245b6503fe961d7992749ed.jpg)

### TrafficMonitor

TrafficMonitor 是一款开源、极简的 Windows 系统资源实时监控工具，它能将当前精准的网速、CPU 与内存占用率极其优雅地嵌入任务栏或悬浮于桌面，让你随时直观掌控电脑的性能状态。

> 下载链接：[TrafficMonitor](https://github.com/zhongyang219/TrafficMonitor/releases)

![](https://img.lunamyth.love/2026/03/3b2bc30311414f6fa6f13340cf8078d9.jpg)

### Clash Verge

Clash Verge 是一款基于强大内核的现代化开源网络代理客户端，它凭借高颜值的 UI 界面与灵活的智能路由规则，让你极其优雅且高效地管理节点订阅并接管全局网络访问。

> 下载链接：[Clash Verge](https://github.com/clash-verge-rev/clash-verge-rev)

![](https://img.lunamyth.love/2026/03/b896f5cecebf7a6a56ae6fbb929ef196.jpg)
