## 前言

人的记忆总是不靠谱，总是会在艳阳高照的下午，听着过去熟悉的旋律，独自黯然神伤。为了避免这样的 Bad End 发生，我决定记录下我每一周都做了什么。有点像写日记——不过个人博客不就是用来做这个的吗？希望有朝一日回首往事的时候，我可以看着过去傻笑，而不是把枕头给弄湿了。

### 第一周

`（2026-03-09 - 2026-03-15）`

有天早上上厕所刷知乎的时候，又刷到了一个关于病毒吧吧主的提问。这就让我了解到了一个我之前没听说过的杀毒软件——[卡巴斯基](https://www.kaspersky.com.cn/)。之前我一直都用的是微软自带的杀毒软件，于是我想看看为什么这个软件会得到大家的好评。

当天下午我就在图书馆玩耍这个软件，最初印象就是 UI 设计特别好看，然后我就遇到了一大堆麻烦事。比如说它为了更安全的保护我们的电脑，推出了一个适用于浏览器的插件。我装了那个插件之后，结果连机场网站都打不开了。很是郁闷，果断卸载了这个插件。结果还是各种屏蔽，我就开始在软件的各个设置里面一个一个排查究竟是谁搞的鬼。最终把安全浏览关了，后面测试发现付款也有保护，我就再把反钓鱼关了，这下舒服多了！

![](https://img.lunamyth.love/2026/03/c5bdf6727a5d97a84200e5671e86dd76.jpg)

还有一个关于这个杀毒软件的故事，发生在我们这学期开设的数据库课上。老师给我们发了一个软件，说是之后的作业、练习、资料之类都可以在上面查看。但是啊，老师发的这个软件有   synaptics 病毒！这就迫使我学习了该如何使用卡巴斯基隔离文件或者文件夹。就在我写这段话的时候，我又去尝试了一下，发现依旧有病毒！就这样写到一半突然没法操作电脑了，因为卡巴斯基正在深度清除。我之后决定把这个软件放在虚拟机下，看看能不能成功。

说的有点远了，回到正题。那时我正在玩耍卡巴斯基，突然想到我之前遇到的一个 bug，就是如果用系统自带的程序来解压系统自带的程序压缩的文件，有些时候会失败。正好我买了 gemini pro，不用白不用:spoiler[（这一周正好到了续费的时间，我脑抽直接买了一年，还是代充，不过那个网站有 ICP 备案，应该还好吧）]。一问才知道，如果路径名不能太长了。我又想到我平时用的什么 7-Zip，什么 WinRAR，界面都丑的很，不喜欢。通过 gemini 了解到可以使用 NanaZip，下载用了一下，确实好用。再用 geek 把那些不用的解压软件卸载了。

正在我享受好看的图标的时候，一不小心卸载了一个我过去留下的痕迹——高中第一次混百合圈，在 S1 活动的记录。而且我还特别顺手的把回收站也给清除了。我去，我一下子慌了，我快速在脑海中想该怎么办。我记得我之前看到一篇知乎回答，讲的是为什么 steam 卸载一个几十个 G 的游戏这么快。那个人说是因为那还不是真正的删除，只是标记了这块地可以用，原来的数据依旧在。我突然找到了一点转机，开始上网搜索数据恢复。试了一下，没有一个软件都没能找到我的那个文件。AI 给我推荐了几个微软商店上有的数据恢复软件，不过还是没能成功。我无意中看到一个名字很像广告，但是评分还挺高的软件。我去，真的找到了，我好激动。

![](https://img.lunamyth.love/2026/03/2f742e3971e6c2dc01443e29b4720250.jpg)

看到那个熟悉的文件名，我一下子点进去，发现大大的几个字——收费！单个文件三十多，或者花一年买一个终身会员。我生气了，准备再去找找有没有其他可以使用的软件。我因为觉得那个很像广告的软件可能会让我的电脑中病毒，于是就用 geek 强力卸载了。接着我继续来找还有没有合适的数据恢复软件。

我最终都没找到有用的，当时想的是大不了就是给个一百多终生会员，之后可以一直用也不错。没想到啊，我这次竟然没找到我的那个文件。我不信，又重新跑了好多遍，结果都没有。我有点着急了，打开了深度恢复，扫描了几百个 G 的文件，终于找到了那个文件！不过啊，那个文件竟然坏了，用不了了。我似乎突然意思到了我在做什么，我在数据消失的 C 盘反复增删文件，所以原来照片的数据被覆盖了。至于为什么是 C 盘，因为微软商店默认的下载路径就是这个……

![](https://img.lunamyth.love/2026/03/989043af66c8bd1eafa72480499a3585.jpg)

无奈之下用 Everything 搜了一下那个我的那个文件的名字，结果还真有备份。呜呜，真是命运弄人。这次也算是花钱买了一个教训，下一次我一定把重要数据保存在网盘里面。还有就是不要在找回文件的过程中玩电脑了。

:::WARNING
恢复数据的时候一定不能再玩电脑了！
:::

有天早上，我正在日常和 AI 聊天。我一直都想重新再做一个个人博客，这个博客要美观，而且对 Markdown 和 Latex 的支持一定要非常好。我就问 AI 做一个个人网站需不需要 JavaSpring 的知识，它说不用。这突然让我有了兴趣，便继续与 AI 对话。我这次不想用 github.io，因为我之前的体验很不好，便突发奇想买一个服务器来玩玩。我打开阿里云，完成了学生认证，送了一张 300 远的优惠券。但是发现能只有为数不多的服务器可以用这个，而且还是按量计费，不能 ICP 备案。最后选了一个新用户优惠 99 元一年，2 核 2 G 内存。

![](https://img.lunamyth.love/2026/03/a2d9cb19e4daea059399a4b457dd2894.jpg)

这也是我的第一个服务器。不过我最终还是没有把这个服务器用于部署我的个人网站，不是嫌麻烦，主要是我突然意识到：我的博客不可避免会说很多百合相关的话题，如果放在国内的话，说不定哪一天就把我给干掉了。再三思考之下，决定使用 AI 推荐的 Cloudflare Page。所以目前这台服务器暂时是闲置状态。根据 AI 的建议，我在上面装上了 [ShellCrash](https://github.com/juewuy/ShellCrash)，才发现在 Linux 系统上翻墙这么简单 :spoiler[（Windows Terminal 下配置终端颜色真好看！）]。

![](https://img.lunamyth.love/2026/03/1b8067515577fa15486d200450c2ef7e.jpg)

在询问 AI 的建议之后，我用了一个名为 [Fuwari](https://github.com/saicaca/fuwari) 的框架:spoiler[（不管从什么角度来说都非常适合我的博客风格啊！）]。把仓库克隆在本地之后，我就一直在玩这个玩具，玩各种配置，期间我还去成功获得 Banner 图片的授权。初步熟悉之后，便是那个我久闻大名的 Cloudflare。其实我很早之前就知道了它，因为在几年前，我电脑翻墙就是用的这个，不仅免费，而且速度还快。不过现在早就不能用了。

我把 github 仓库绑定 Page，我问 AI 下一步该做什么，它说这样就行了！啊，如此简单，如此便捷，而且还是免费的！我突然就理解了互联网上大家为什么对 Cloudflare 的评价如此高。之前我在阿里云买的是一个 .cn 域名，风险实在是太大了，我于是就想买一个新的域名。我决定来一个 Cloudflare 全家桶，便在这上面买域名，发现没有中国的支付方式。问了一下 AI，了解到 Paypal 可以绑定中国的银行卡，我便开始拿出那个不知道被我放在哪的建行卡，没想到真的成功了。我很开心，到了支付页面，发现最后一步确认支付总是报错，上网寻找解决方法无果，那时候已经是凌晨两三点了。

我看遇见同样问题的人在论坛里面讨论，其中有一个人提到了一个网站 [SpaceShip](https://www.spaceship.com/zh/)，我便抱着试一试的心态进入了这个网站。没想到真的可以买，而且还不贵，甚至可以用支付宝直接付款。果断购买，然后用 Cloudflare 管理这个域名，和我的仓库绑定。最终，在凌晨 4 点，我终于可以用 [lunamyth.love](lunamyth.love) 了，网页图标就是那个可爱的🐡！

![](https://img.lunamyth.love/2026/03/afaaeb28003d8fd699a04ac1a545015d.jpg)

用 Cloudflare 部署网页特别方便，我只要 push 到 github 之后，就能实时部署，完全不用在意网络问题。于是我继续寻找 Cloudflare 上面还有什么好玩的东西，了解到可以用它的 R2 对象存储做图床。于是我在某天晚上和我妹妹玩完泰拉瑞亚之后开始做这个，那时都凌晨了。通过 AI 的建议，我使用了 [PicList](https://piclist.cn/)，稍微配置一下就能用，而且 UI 特别好看。Cloudflare 真好用啊，之后如果我可以支付了，我一定要买 pro 支持一下。

![](https://img.lunamyth.love/2026/03/f182946d52a104fa10101e00dc7ffadf.jpg)

这周我写了第一篇文章，讲的是软件分享，我一直都很想写这个。我也在知乎上发布了我的文章，现在已经有 10 赞同，44 收藏了，很不错！这周总体上来说学到了很多，但是有两个不足：一是没有时间学英语，二是每天晚上的入睡时间太晚了，希望下一周可以做出一些调整与改变。对了，我终于知道该调整适配 Maple Mono 字体的 VSCode 的句子分行了，只用改一下这个配置。

![](https://img.lunamyth.love/2026/03/b0d9551510273ab754373201dd03ef45.jpg)

我开始在 MDN 上复习之前学到的前端知识，为我之后开发网站做准备。那我就以我昨天晚上康复训练的静态网页作为结束吧，下周见！:spoiler[忙得都没时间看明日酱了，呜呜……]

![](https://img.lunamyth.love/2026/03/e2f3f77e733e8b6fc0246b79134ef6ff.jpg)

### 第二周

`2026-03-16 - 2026-03-22`

就在刚刚，我发现原来这周做了这么多的事，这种对时间的感觉或许就是这篇文章存在的价值吧。

首先就是上一周的后续，当时我不是还在处理那个老师发过来的有病毒的文件吗，最后我准备采取的方法就是关在虚拟机里面。对不起，卡巴斯基，原来这个软件真的病毒得不能再病毒了，连微软自带的杀毒软件都能立马检测到。然受就是我发现虚拟机的快照功能真的好好用，可以管理不同的分支。于是我一个分支用老师的病毒软件，一个分支用完美 VSCode 配置，非常不错！

![](https://img.lunamyth.love/2026/03/04017f4bccc9fc395166065cefa26eeb.jpg)

我经常熬夜，使得我星期二的早八差点就迟到了:spoiler[已经迟到了]。好在老师为人比较温和，没有计较我突然闯入教室。当时我在刷知乎，想看一看为什么我的 Paypal 不能付款，大家都说不要翻墙注册账号。我说这下完了，以后我想买 Cloudflare 上的服务咋整。我抱着侥幸的心态再来试一试，反正到时候应该也有确认支付，钱不会马上扣除。于是我就尝试买一个新域名 `seabed.love`，刚点击支付，然后就自动扣款成功了？我喜出望外，没想到这么简单就成功了。现在回想起来的话，可能和我绑定的那张建行银行的卡没钱了有关。既然这有了新域名，那肯定不能闲着，我目前的打算就是用它部署一个百合小说漫画在线观看，游戏动画在线下载的这样一个网站。不过我的能力还不够，现在只是先用这个域名绑定一个专门用于分享文件的图床。我学会了使用 [Cyberduck](https://cyberduck.io/) 来管理文件，虽然没有 Piclist UI 界面那么好看，但是功能十分强大。

![](https://img.lunamyth.love/2026/03/74140512d76159772b5d900ada1c56a7.jpg)

我用 Cyberduck 遇到的一个很大的问题就是，它不支持用自定义域名来复制下载链接。我想到了 Piclist 强大的复制链接功能，于是尝试看看用它来管理行不行。发现有三个问题，一个是图床冲突了，我当时的配置使得复制文件链接的时候依旧保持着原来的链接；第二个是上传文件之后的预览图很丑，话说文件本来就不应该有预览图；最后一个是它甚至不能修改文件存放路径，这一点是最严重的问题。面对这些问题，必须要 Cyberduck 和 Piclist 混合使用才行，通过询问 Gemini，我了解到 Piclist 上图床选项卡和云端选项卡代表的含义，原来云端可以管理多个图床，每个图床个性化的设置在图床选项卡里面。成功配置之后，再加上 Cyberduck 强大的文件管理能力，终于完成了最舒服的图床配置！

![](https://img.lunamyth.love/2026/03/60624af22910d32c82acc0ad4b1964b7.jpg)

我在 yuri-garden 这个图床里面上传了一些使用的软件压缩包，我也第一次直观的感受到 .7z 格式巨大的压缩率。这里面有一个小插曲，就是我发现我刚刚在图床更新的新文件，竟然通过链接去下载还是下载的老文件。通过 AI 我才了解到这是缓存在搞的鬼，了解到了什么是浏览器 TTL 和边缘 TTL，最后通过自定义清除顺利解决。当然了，之后最佳的方法还是给文件名加一个版本号或者内容描述。

![](https://img.lunamyth.love/2026/03/6c37effe862f5b27adfff31e34ea5a56.jpg)

我无意之间看到 QQ 中一个人的个人简介，发现她竟然使用了自定义域名作为邮箱。这不是很好玩吗，于是就问 AI 现在主流的实现方法是什么。它告诉我用 Cloudflare 接收邮件，用 [Resend](https://resend.com/emails) 发送邮箱。我根据步骤一步一步进行配置，果然成功了！

![](https://img.lunamyth.love/2026/03/56f8c1fd1512677d094fa835c619d25c.jpg)

![](https://img.lunamyth.love/2026/03/4971ece5f1aac4ef37ef786832844e7e.jpg)

最后我还用 [mail-tester](https://www.mail-tester.com/) 测试了一个邮件质量如何，成功获得 10 分！欢迎大家通过这个邮箱来 `saluton@lunamyth.love` 找我玩哦！

![](https://img.lunamyth.love/2026/03/d323ecf504608fbaffa5403e9309860f.jpg)

这周还有一个大事就是我发现 AI 好像在推荐软件这方面非常擅长，而我之前对那些远程控制的软件非常不满意，于是问了一下 AI。它给我说可以使用 [Sunshine](https://app.lizardbyte.dev/Sunshine/?lng=zh-CN) 作为主机，使用 [Moonlight](https://moonlight-stream.org/) 作为客户端，非常好用！不收费，可以调任意分辨率，低延迟，完美模仿触控板来实现鼠标移动。我觉得串流非常好玩，虽然当天中午就在宿舍床上用手机串流电脑看明日酱的水手服睡着了。

但是呀，手机屏幕就那么大，而且我还经常把手机的钢化膜摔碎，用来串流玩视觉小说的感觉也太不好了吧。我于是决定购入一台平板，能够满足我串流玩视觉小说，完美谷歌的需求，大屏看百合漫画动画的需求。通过 AI 最终选中了这款[Xiaomi Pad 7 Pro](https://www.mi.com/prod/xiaomi-pad-7-pro)。我去京东上看价格，发现现在正好是国补，折扣下来一千七。下面是它的配置，:spoiler[存储空间果然是最不重要的]。

![](https://img.lunamyth.love/2026/03/59fe57044f3040c05272e53e2544cb3c.jpg)

由于这一次装软件已经是轻车熟路了，所以我顺利的 Google，删除了多余的系统自带软件，装上了必要的各种软件，配置好了各种环境，终于有了一个非常舒服的桌面。不得不提在大屏幕就是舒服，最适合来玩那些视觉小说了。我顺路再买了一个显卡诱骗器，这下可以安心的把笔记本盖给扣上了，嘻嘻。

![](https://img.lunamyth.love/2026/03/cdd3a0d79bf6c092af293b1b47ec7a55.jpg)

平板上面打开 Chrome 的体验简直和 PC 端一模一样，可以非常丝滑地在多个标签页之间互切。有一天我下载了一个 .json 文件，但是在手机上无法正常打开这个文件。我说这个文件不是和文本文档差不多性质的吗，竟然还无法打开。我就问 AI 手机上有没有像 VSCode 一样的软件，可以打开这些文件。AI 给我推荐了一个用 [Termux](https://github.com/termux) + [code-server](https://github.com/coder/code-server) 实现的方法，我再把这个 code-server 添加到桌面，通过离线安装插件的方法安装插件（甚至要用拼接链接的方式实现插件的下载），最后竟然实现了和电脑上 VSCode 差不多的效果。这还是我认识的安卓吗，太神奇了！这也是我第一次用 [Clangd](https://clangd.llvm.org/) 来编译 C++。

![](https://img.lunamyth.love/2026/03/bb6ee94ca3046ca1aca801f72f109397.jpg)

这周我继续在 MDN 上学习前端知识，不过遗憾的是只坚持了两三天的英语学习，并且我没有一天成功早睡。非常疲惫的一周，花在配置和装饰上的时间绝对比学习的时间多很多。希望我下一周可以有所改变。你看我连明日酱的水手服都才看到第三集，太忙碌了……

### 第三周

`2026-03-28 - 2026-03-29`

可能因为我的不规律作息，我的身体已经十分疲惫，疲惫到无法构建出正常的梦境。于是在有一天下午放学的时候，我果断选择回宿舍，决定在晚上七点钟就早早入睡。不过中途因为学校归寝签到醒了一次，也因此困意消却了不少。不想回到床上，我便拿出平板串流开始重温 SeaBed 。也得益于此，我发现了我平时写文与 SeaBed 最大的不同，那就是我情感表达的部分远多于具体记录，简单来说就是我缺少一种用简明清晰的语言表述一件事物的能力。

![](https://img.lunamyth.love/2026/03/188afd7d5394726f37c4b2af855eb83d.jpg)

高中的时候我也喜欢写日记，不过因为我无事可做的懒惰，导致了无事可写的空白。我那个时候喜欢一长段的自我心理刻画，越写越伤心，最后还自顾自的哭了起来。现在回想起当初的事，我只能如此评价我自己：「放大甚至幻想自己的痛苦，从而感到心满意足，尝试构建一个悲剧主角的形象」。日记的作用是供我们日后回忆当初，所以重在记事。现在呢，我就开始这周的记事。

这一周从表面上来看我做的事情没有上两周多，我认为这是因为「树上低垂的果实已被尽数采摘」，现在到达的一个平稳的发展阶段。我比较高兴的事情是，这周是我进入大学以来背单词最多的一周。这有两个原因，第一个是因为改用了平板来背单词，大屏幕使人心情愉悦；第二个是我认识到不能够把碎片时间用于背单词（:spoiler[也不能用碎片时间看百合，百合可是要舒舒服服待在一个安静的地方慢慢来欣赏的！]），因为把缓冲时间的放松变得焦虑只会导致双输的结果。目前我欧路词典的背单词规划是 50 + 150，也就是每天 50 个新词的学习和 150 个旧词的复习。我也买了一本 4 级写作，下一周开始我要开始背范文，这对我来说很有难度，因为我的记忆力不好。我也要找时间做一下四级真题，看看我背的单词到底有没有起到作用。

![](https://img.lunamyth.love/2026/03/70719c4a396062838dbb9e2310bc7376.jpg)

我还有一个每天坚持做的事情就是在 MDN 上面学习前端知识，真的很有意思，比我之前学 C++ 和 Python 的时候要有趣得多。我觉得我自己可能对网络方面更感兴趣，更喜欢跟浏览器打交道。不过呀我的进度确实比较忙，因为最近的事情也比较多，我目前刚刚学到 css 的盒模型。阶段性目标的话，我想做一个在线的 markdown 编辑器，希望有朝一日可以实现。

这一周差不多就这样短短结束了，我希望下一周我依旧可以保持每天早睡，学英语的好习惯。我也采购了一个床上台灯，疲惫之余窝在床上看漫画或者玩视觉小说都是一个非常不错的选择！还有啊，就是我的体重已经比我刚开学的时候减轻了 10 斤，这可能是因为我之前太累了吧。
